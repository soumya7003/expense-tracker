# Expense Tracker

A simple yet powerful personal finance dashboard built with vanilla JavaScript, HTML5, and CSS3. Track your income and expenses, visualize spending trends, and stay on top of your budget — all in the browser, with no backend required.

🔗 **Live Demo:** [expense-tracker-logixor.netlify.app](https://expense-tracker-logixor.netlify.app/)

## Features

- 📊 **Real-time Analytics** — Visualize income, expenses, and spending trends using Chart.js
- 💱 **Currency Conversion** — Built-in support for converting amounts across currencies
- 🌗 **Dark / Light Theme Toggle** — Switch between themes for comfortable viewing
- 🔍 **Advanced Search & Filtering** — Quickly find transactions by category, date, or type
- 💾 **LocalStorage Persistence** — All data is saved locally in the browser, no sign-up needed
- 📤 **One-Click CSV Export** — Export your transaction history for offline use or backup

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
- Chart.js

## Getting Started

### Prerequisites

You only need a modern web browser. No build tools or dependencies required.

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/soumya7003/expense-tracker.git
   ```
2. Navigate to the project directory
   ```bash
   cd expense-tracker
   ```
3. Open `index.html` in your browser
   ```bash
   open index.html
   ```

That's it — no installation or server setup needed!

## Usage

1. Add a new transaction (income or expense) with a category and amount.
2. View your updated balance and spending breakdown instantly via the dashboard charts.
3. Use the search/filter bar to locate specific transactions.
4. Toggle between dark and light mode using the theme switch.
5. Export your transaction history as a CSV file whenever you need it.
